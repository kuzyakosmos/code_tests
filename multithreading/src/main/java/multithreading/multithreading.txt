JVM завершает процесс только тогда, когда не осталось ни одного non-daemon потока.
main — это обычный non-daemon поток, не особенный.

Что такое daemon-поток - Фоновые, сервисные потоки
JVM не ждёт их завершения
Примеры:
GC
ForkJoinPool common
watchdog-и - это механизм наблюдения, который контролирует работоспособность системы
